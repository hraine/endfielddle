# 🎮 Emoji Game - Руководство

## ✨ Что это

**Emoji Game** - это daily guessing режим в ENDLE, где нужно угадать персонажа по эмодзи. Каждый день - новый персонаж!

## 🎯 Как играть

1. **Выбери режим "😊 Emoji Game"** на главном экране
2. **Смотри на эмодзи** - они подсказывают персонажа
3. **Начни вводить имя** - появится автодополнение
4. **Выбери из списка** или введи полностью
5. **Используй подсказки** если нужно (💡 Hint)
6. **У тебя 8 попыток** чтобы угадать!

## 🔍 Автодополнение

При вводе имени персонажа автоматически появляется список подходящих вариантов:
- Начни вводить имя (например: "Ard")
- Появится список персонажей
- Используй ↑↓ для навигации
- Нажми Enter или кликни для выбора

## 💡 Подсказки

Доступны 3 уровня подсказок:
1. **Hint** - описание персонажа (если добавлено)
2. **First letter** - первая буква имени
3. **Length** - количество букв в имени

## ⚙️ Управление персонажами

### На главном экране:
- Нажми кнопку **"⚙️ Manage Emoji Characters"**
- Откроется панель управления

### Добавление персонажа:
1. **Emoji**: Введи эмодзи (например: 🦊🔥)
2. **Name**: Имя персонажа (например: Ardelia)
3. **Hint**: Подсказка (необязательно)
4. Нажми **"Add Character"**

### Удаление персонажа:
- В списке персонажей нажми **"Delete"** рядом с нужным

## 📅 Daily System

- **Один персонаж в день** - как в Wordle
- **Seed-based** - у всех игроков одинаковый персонаж
- **8 попыток** на угадывание
- **Прогресс сохраняется** в localStorage

## 🎨 Советы по созданию эмодзи

1. **2-3 эмодзи** оптимально
2. **Узнаваемые символы:**
   - Животные для персонажей-животных
   - Оружие для воинов
   - Элементы для магов
3. **Комбинации:**
   - Персонаж + элемент (🦊🔥)
   - Класс + оружие (⚔️👸)
   - Символ + способность (🛡️🔥)

## 💾 Хранение

- **localStorage** - все персонажи сохраняются локально
- **Ключ**: `endle_emoji_characters`
- **Формат**: JSON массив
- **Автосохранение** при добавлении/удалении
